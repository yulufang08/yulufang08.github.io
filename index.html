🔬 FinAgent - Industrial Research AI

Multi-agent autonomous industry analysis system

Nov. 2025 – Feb. 2026

LangGraph powered multi-agent collaboration with hybrid retrieval, real-time web search, and deep analysis for autonomous industry research.

Key Features

  • LangGraph state machine with Self-Correction loops
  • Hybrid retrieval: BM25 + Vector + RRF fusion
  • Real-time web search with source authority verification
  • Multi-dimensional analysis: supply chain, competition, finance
  • Full source traceability with snippet IDs
  • Free Groq API integration (unlimited tokens)

⚡ 如何使用 Web UI

  1. 打开终端，进入项目目录
  2. 运行：python finagent_web.py
  3. 点击下方"🚀 启动 Web UI"按钮
  4. 在弹出窗口中输入 API Key 开始使用
