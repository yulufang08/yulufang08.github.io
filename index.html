🔬 FinAgent - 自主化产业研究系统

一个智能产业分析工具，结合 LangGraph 多 Agent 协作、混合检索、联网搜索和深度分析。 自动完成行业研究、数据收集、结构化分析和报告生成。

核心特性：

  • ✨ LangGraph 状态机 - 支持 Self-Correction 循环的 Agent 协作
  • 🔍 混合检索 - BM25 + Vector Search + RRF 融合
  • 🌐 联网搜索 - 实时获取最新行业数据，注重信源权威性
  • 📊 多维分析 - 产业链、竞争格局、财务数据等深度分析
  • 📄 结构化报告 - 每个结论附带来源追溯 [snippet-id]
  • 🔐 身份认证 - 仅授权用户可使用（保护 Token 消耗）

💡 使用建议

  • 选择正确的关键词：输入具体的行业名称或产品类别（如"固态电池"、"AGI芯片"）
  • 合理设置迭代次数：1-2次用于快速查询，3-5次用于深度研究
  • 关注数据来源：每个结论都标注了来源，可点击查看原始片段
  • 激活联网搜索：检查框以获取最新信息，离线模式使用本地文档
  • 注意 Token 消耗：深度研究会消耗更多 Token，请合理规划使用

⚠️ 重要提示

Web UI 仅供本人使用，需要 API Key 认证。其他访问者可以在 GitHub 上查看源码、学习架构设计和部署指南。 不提供公开的在线演示版本，以保护 API 密钥安全和控制 Token 成本。
